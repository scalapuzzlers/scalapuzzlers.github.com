What is the result of executing the following code?

var IJ: (Int, Int) = (3, 4)
var (I, J): (Int, Int) = (3, 4)
  1. Prints: 
    IJ: (Int, Int) = (3, 4)
I: Int = 3
J: Int = 4
  2. Both fail with compilation exceptions
  3. The first one succeeds and prints: 
    IJ: (Int, Int) = (3, 4)
    and the second one fails with a compilation exception
  4. The first one succeeds and prints: 
    IJ: (Int, Int) = (3, 4)
    and the second one fails with a runtime exception

Explanation

The former is an assignment of a tuple which is type correct. The latter tries to match (3, 4) against two constants I and J â€“ Scala assumes that UPPERCASE variables in pattern matches are constants.
