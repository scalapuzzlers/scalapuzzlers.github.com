What is the result of executing the following code?

var IJ: (Int, Int) = (3, 4)
var (I, J): (Int, Int) = (3, 4)
  1. Prints: 
    IJ: (Int, Int) = (3, 4)
I: Int = 3
J: Int = 4
  2. Both statements fail to compile
  3. The first statement prints: 
    IJ: (Int, Int) = (3, 4)
    and the second fails to compile
  4. The first statement prints: 
    IJ: (Int, Int) = (3, 4)
    and the second one throws a runtime exception

Explanation

The former is an assignment of a tuple which is type correct. The latter tries to match (3, 4) against two constants I and J â€“ Scala assumes that UPPERCASE variables in pattern matches are constants.
