What is the result of executing the following code?

object TwoD {
  case class Point(x: Int, y: Int)
  object Point {
    def ->(coords: (Int, Int)) = apply(coords._1, coords._2)
  }
}

val origin = TwoD.Point -> 0 -> 0
val zeroone = TwoD.Point -> 0 -> 1
println(origin.x == zeroone.x)
  1. Prints: 
    true
  2. Prints: 
    false
  3. Fails to compile
  4. Throws an exception
