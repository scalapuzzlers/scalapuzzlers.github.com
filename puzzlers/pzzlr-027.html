What is the result of executing the following code?

val sb = new StringBuilder("Hello, kitteh!")
val width = 20
1 to width - sb.length foreach sb.append(' ')
println(s"[$sb]")

Explanation

  

Recall that Scala has a
StringBuilder that

shadows java.lang.StringBuilder

and that it has an

apply

method.
  

  

The foreach takes a function with an Int parameter.
StringBuilder.append returns a StringBuilder
which, as a Seq, is such a function; that function is eta-expanded.
The append is evaluated once at eta-expansion;
each iteration invokes StringBuilder.apply, which is an
alias for StringBuilder.charAt.
If the StringBuilder had not been initialized with data,
that invocation would throw StringIndexOutOfBoundsException.
