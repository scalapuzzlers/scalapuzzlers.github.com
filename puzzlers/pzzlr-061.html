What is the result of executing the following code?

def firstElement[T](seq: Seq[T]): Unit =
  seq match {
    case x :: xs => println(x)
    case _ => println("Nothing")
  }
val seq1: Seq[Int] = Seq(1,2,3)
firstElement(seq1)
  1. 1
  2. Nothing
  3. 2
  4. 3

What is the result of executing the following code?

import java.util.LinkedList
import scala.collection.JavaConversions._

val jList = new LinkedList()

jList.add(1)
jList.add(2)
jList.add(3)

val seq2 = jList.toSeq
firstElement(seq2)
  1. 1
  2. Nothing
  3. 2
  4. 3

Explanation

The frenquently used type Seq is an interface with an apply method that returns a List when called so seq1 is List(1,2,3) therfore matches the first case. The java conversion method that's invoked when we call toSeq on a LinkedList retuns a Buffer which does not match the first case (only scala List have a :: defined on them). To catch all sequences we should have used the +: for the matching case.
