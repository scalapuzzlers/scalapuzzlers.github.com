Tested with Scala version 2.10

What is the the value of allTogether and allTogether2 ?

//each map has names and number of names.
val names = Map("Sidney" -> 1 , "Paul" -> 1 , "Jacob" -> 7, "Bob" -> 12)
val moreNames = Map("Sidney" -> 1 , "Paul" -> 5 , "Nick" -> 2)

val allTogether = names ++ moreNames
val allTogether2 = moreNames ++ names
  1. allTogether = Map(Bob -> 12, Sidney -> 2, Nick -> 2, Jacob -> 7, Paul -> 6)
    allTogether2 = Map(Bob -> 12, Sidney -> 2, Nick -> 2, Jacob -> 7, Paul -> 6)
  2. allTogether = Map("Sidney" -> 2 , "Paul" -> 6 , "Jacob" -> 7, "Bob" -> 12)
    allTogether2 = Map("Sidney" -> 2 , "Paul" -> 6 , "Nick" -> 2)
  3. allTogether = Map(Bob -> 12, Sidney -> 1, Nick -> 2, Jacob -> 7, Paul -> 5)
    allTogether2 = Map(Bob -> 12, Sidney -> 1, Nick -> 2, Jacob -> 7, Paul -> 1)
  4. Error : Cannot resolve symbol ++

Explanation

Map1 ++ Map2 replaces any (k,v) from the Map1 with (k,v) from Map2. if key isnt in Map2 will leave Map1 as is.
